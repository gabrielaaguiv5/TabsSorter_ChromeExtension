# Tabs Sorter - Chrome Extension

Tabs Sorter is a Chrome extension that displays a list of all your currently open tabs on every new tab page.  
You can instantly switch to any tab by clicking on it.

Designed to help users manage many open tabs quickly and efficiently.

---

## Features

- Shows all open tabs in one place
- Displays tab titles clearly
- One-click navigation to any tab
- Works automatically on every new tab
- Lightweight and easy to use

---

## Premium Features (Subscription)

Tabs Sorter is free to use with basic tab listing and navigation.

A Premium subscription unlocks additional features designed for heavy tab users:

- Alphabetical tab sorting
- More advanced organization options:
  - Last used
  - Most recently opened

Subscriptions help support development and ongoing improvements!

---

## How to Use
1. Install the extension
2. Open multiple browser tabs
3. Open a new tab
4. View the list of all open tabs
5. Click any tab to switch to it instantly

---

## Permissions and Privacy
This extension uses Chrome tab permissions only to:
- Read the list of open tabs
- Switch to a selected tab when clicked

No personal data is collected, stored, or shared in the free version.

## Project Structure
